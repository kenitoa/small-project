# Logic Mini Projects

## Overview

이 저장소는 다양한 **로직(Logic)**을 구현하고 실험하기 위한 **미니 프로젝트(Mini Project)**들을 정리한 공간입니다.
각 프로젝트는 비교적 짧은 기간 동안 아이디어를 구현하는 것을 목표로 하며, 프로그램 구조와 로직 설계를 중심으로 구성되어 있습니다.

## Project Characteristics

* 하나의 프로젝트는 **1개의 폴더에 여러 파일로 구성**됩니다.
* 프로젝트의 **구성 기간은 약 1일 ~ 1주일 이내**를 기준으로 진행됩니다.
* 프로그램의 동작 방식과 문제 해결 과정을 중심으로 **주요 로직 구성**에 초점을 맞추고 있습니다.

## Structure

```
mini_project
├─ main
├─ logic
├─ utils
└─ README.md
```

각 프로젝트는 기능별로 파일을 분리하여 관리하며,
프로그램의 핵심 동작은 로직 단위로 나누어 구성합니다.

## Purpose

이 저장소는 다양한 로직 구현을 실험하고 기록하기 위한 프로젝트 모음입니다.
작은 규모의 프로젝트를 통해 문제 해결 방식과 프로그램 구조를 정리하는 데 목적이 있습니다.
